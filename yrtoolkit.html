Tutorial

Click on the Blocks button on the right side top bar.

Click on the Angry button on the left.

Click and drag the "When Angry.Click do" block work space.

Click on the "Angry_number" button on the left, then drag and drop the "Set Angry_number.Text to" block in the block work space. Add the "Set Angry_number.Text to" block to the "When Angry.Click do" block.

Click on the "Math" button in the left column. Click and drag the "+" block and add it to the "Set Angry_number.Text to" block.

Click on the "Angry_number" button in the left column. Click and drag the "Angry_number.Text" block and add it to the first blank in the "+" block.

The blocks should look like.

Click on the "Math" button in the left column. Click and drag the "0" block and add it to the second blank in the "+" block.

Change the value in the "0" block to 1.

Lets add features to the apps

  • Show an inspiring quotw based on how the user feels.

    • Set the label component's text to a random message after the user chooses their mood.

  • Display an image or play a song to cheer up a friend.

    • Use the image component to show a picture, or the player component to play a song after the user selects an emotion.

  • Personalize the app with the user’s name.

    • Use the text input component to get the user’s name and display it in the app.

  • Text a friend when the user is feeling sad.

    • Use the texting component to send a message when the sad button is clicked.

  • Let users review how they felt in the past.

    • You can use the TinyDb component to store what happens each time the user hit the button.
